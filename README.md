## 📌 Descripción

**Proyecto_Colores_Aleatorios** es una aplicación web sencilla que genera colores aleatorios de forma dinámica utilizando JavaScript. Permite cambiar el color de fondo (y opcionalmente mostrar su código) con solo interactuar con la interfaz.

Este proyecto es ideal para practicar conceptos básicos de desarrollo web como manipulación del DOM, eventos y estilos dinámicos.

---

## 🚀 Tecnologías utilizadas

* **HTML** → estructura de la página
* **CSS** → estilos y diseño visual
* **JavaScript** → lógica para generar colores aleatorios

---

## ⚙️ Funcionalidades

* Generación de colores aleatorios en formato:

  * HEX (#FFFFFF)
  * RGB (opcional)
* Cambio dinámico del color de fondo
* Botón interactivo para generar nuevos colores
* Visualización del código del color actual

## 🖥️ Instalación y uso

1. Clona este repositorio:

```bash
git clone https://github.com/tu-usuario/Proyecto_Colores_Aleatorios.git
```

2. Abre el archivo `index.html` en tu navegador.

---

## 💡 Ejemplo de uso

Al hacer clic en un botón como **"Generar Color"**, el fondo de la página cambiará automáticamente a un color aleatorio.
